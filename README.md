Welcome to the official repository of my personal portfolio. This is a **Tier-1 Data Science and ML Engineering portfolio** designed to highlight professional internship experiences, technical public-sector products, academic publications, and end-to-end Machine Learning pipelines.

---

## 🌟 Key Features

*   **Dynamic Bento Grid for Research**: A premium 3-column layout highlighting high-impact academic and applied research projects (such as Temporal Fusion Transformers and credit risk modeling) with distinct visual hierarchies.
*   **Interactive Project Filter Bar**: Built-in state management allowing visitors to filter work dynamically across four categories: **ML/AI**, **Dashboard**, **SaaS**, and **Research**.
*   **Metrics & Stats Strip**: Direct quantitative impact numbers displayed in a sleek numerical banner at the top fold of the page.
*   **Highly Custom Design System**: Pure CSS architecture using semantic custom properties (`:root`), bespoke shadows, state-driven badges, and custom inline SVG branding.
*   **Open to Work Status**: Features a real-time pulsing green status indicator to display immediate career availability.
*   **Full Search Engine Optimization (SEO)**: Custom-configured Open Graph tags, detailed meta descriptions, theme color variables, and responsive titles.

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 18+ is recommended).

### Installation & Run

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/anggatamr/angga-portfolio.git
    cd angga-portfolio
    ```

2.  **Install project dependencies**:
    ```bash
    npm install
    ```

3.  **Start the local development server**:
    ```bash
    npm run dev
    ```
    *Open your browser and navigate to `http://localhost:5173` to see the live app.*

4.  **Produce a production build**:
    ```bash
    npm run build
    ```
    *The output bundle will be generated in the `dist/` directory, optimized and ready for static hosting deployment (Vercel, Netlify, Github Pages, etc.).*
